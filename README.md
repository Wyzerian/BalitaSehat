## Introduction
BalitaSehat adalah aplikasi yang dikembangkan untuk mempermudah proses pengisian dan pengelolaan data anak usia 2 tahun sebagai upaya deteksi dini risiko stunting. Aplikasi ini mengolah data tinggi badan dan berat badan anak untuk menghitung nilai Z-Score berdasarkan WHO Growth Standard, sehingga mampu mengklasifikasikan status pertumbuhan anak ke dalam kategori sehat/normal, stunting, berisiko stunting, serta underweight atau overweight sebagai bentuk early warning system bagi orang tua dan tenaga kesehatan.
